# AutoFishing
Fishing auto-complete.

## Installation (Non-Steam Windows)

1. Download the mod zip from Releases page
2. Navigate to your game's `Duckov_Data` folder
3. Create `Mods` folder (if missing)
4. Extract zip to `Mods` folder
5. Final path should contain all DLL files:
Duckov_Data/Mods/BetterDuckov_v1.10/
- AutoFishing.dll
- 0Harmony.dll
- MonoMod.RuntimeDetour.dll
- MonoMod.Utils.dll

6.Launch game → Mods menu → Enable mod
